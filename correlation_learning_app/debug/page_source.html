Structured Learning Approach

This game is designed to help you learn about correlations through a progressive approach:

1. Read the variable names and think about what relationship you expect

2. Enter your correlation coefficient (r) in the sidebar

3. Click 'Generate Plot' to see your prediction visualized (the number of dots corresponds to the sample size reported in the literature, e.g., Meyer et al., 2001. Sometimes K (# of studies) is given when N is not available)

4. Adjust your guess if needed, then submit for feedback

5. See how close you were to the real correlation from research!



Understanding Correlation Coefficients

Correlation coefficients (r) range from -1 to +1:

  • r = 0: No linear relationship
  • r = ±0.1: Weak relationship
  • r = ±0.3: Moderate relationship
  • r = ±0.5: Strong relationship
  • r = ±0.7: Very strong relationship
  • r = ±1.0: Perfect linear relationship

Note: The scatter plots show standardized variables (z-scores), so the scale is in standard deviation units. Even small correlations can be meaningful in large samples.


Formula:

r = Σ[(x - x̄)(y - ȳ)] / √[Σ(x - x̄)² × Σ(y - ȳ)²]

Example Calculation:

Let's calculate the correlation for a simple example with 5 data points:

Point X Y X - X̄ Y - Ȳ (X - X̄)(Y - Ȳ)
1 1 2 -1 -1 1
2 2 4 0 1 0
3 3 3 1 0 0
4 4 5 2 2 4
5 5 6 3 3 9

X̄ = 3, Ȳ = 4

Numerator: Σ[(x - x̄)(y - ȳ)] = 1 + 0 + 0 + 4 + 9 = 14

Denominator: √[Σ(x - x̄)² × Σ(y - ȳ)²] = √[15 × 15] = 15

r = 14/15 = 0.933
